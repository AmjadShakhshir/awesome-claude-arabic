<hr>

<h2>ما هي مهارات Claude؟</h2>

<p>مهارات Claude (Claude Skills) هي ملفات <code>.md</code> تحتوي على تعليمات نظام مخصصة توضع داخل مجلد <code>.claude/skills/</code> في مشروعك. عند استخدام Claude في ذلك المشروع، يستطيع قراءة هذه المهارات وتطبيقها تلقائياً.</p>

<p>النتيجة: لا تحتاج لكتابة تعليمات طويلة في كل مرة — المهارة تتكفل بذلك.</p>

<h2>كيف تثبّت مهارة؟</h2>
<ol>
  <li>أنشئ مجلد <code>.claude/skills/</code> في جذر مشروعك (إذا لم يكن موجوداً)</li>
  <li>انسخ محتوى ملف المهارة من الصفحة المقابلة</li>
  <li>احفظه باسم مناسب مثل <code>arabic-content-creation.md</code></li>
  <li>ابدأ محادثة Claude في هذا المشروع — المهارة ستكون متاحة</li>
</ol>

<h2>What are Claude Skills?</h2>

<p>Claude Skills are <code>.md</code> files containing custom system instructions placed inside a <code>.claude/skills/</code> folder in your project. When using Claude in that project, it can read these skills and apply them automatically.</p>

<p>The result: you don't need to write long instructions every time — the skill handles it.</p>

<h2>How to install a skill</h2>
<ol>
  <li>Create a <code>.claude/skills/</code> folder in your project root (if it doesn't exist)</li>
  <li>Copy the skill file content from the corresponding skill page</li>
  <li>Save it with an appropriate name like <code>arabic-content-creation.md</code></li>
  <li>Start a Claude conversation in that project — the skill will be available</li>
</ol>
