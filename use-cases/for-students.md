<h1>🎓 الذكاء الاصطناعي للطلاب</h1>

<p>دليل الطالب لاستخدام الذكاء الاصطناعي بطريقة تُعمّق الفهم وتوفر الوقت — لا تستبدل التعلم الفعلي.</p>

<div class="lang-tip">
<p>⚠️ <strong>ملاحظة مهمة:</strong> الذكاء الاصطناعي يساعدك على التعلم بشكل أفضل — ليس على الغش. استخدامه لتسليم عمل ليس عملك يضر بك على المدى البعيد ويخالف قواعد الأمانة الأكاديمية.</p>
</div>

<hr>

<h2>ماذا يحتاج الطلاب عادةً؟</h2>

<ul>
  <li>فهم مفاهيم صعبة بطريقة مبسطة</li>
  <li>تلخيص نصوص ومصادر طويلة</li>
  <li>التحضير للامتحانات باختبار الفهم</li>
  <li>مراجعة الكتابة الأكاديمية وتحسينها</li>
  <li>بناء خطط دراسية واقعية</li>
  <li>استيعاب محتوى بلغة ثانية بشكل أسرع</li>
</ul>

<h1>🎓 AI for Students</h1>

<p>The student's guide to using AI in a way that deepens understanding and saves time — without replacing actual learning.</p>

<div class="lang-tip">
<p>⚠️ <strong>Important note:</strong> AI helps you learn better — not cheat. Using it to submit work that isn't yours harms you in the long run and violates academic integrity policies.</p>
</div>

<hr>

<h2>What students typically need</h2>

<ul>
  <li>Understanding difficult concepts in a simplified way</li>
  <li>Summarizing long texts and sources</li>
  <li>Exam prep by testing understanding</li>
  <li>Reviewing and improving academic writing</li>
  <li>Building realistic study plans</li>
  <li>Absorbing content in a second language faster</li>
</ul>

<h2>Right use vs. wrong use</h2>

<table>
  <thead>
    <tr>
      <th>Right use ✅</th>
      <th>Wrong use ❌</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Asking for explanations in different ways</td>
      <td>Asking for the answer without trying</td>
    </tr>
    <tr>
      <td>Requesting review questions then answering yourself</td>
      <td>Requesting model answers directly</td>
    </tr>
    <tr>
      <td>Reviewing an essay you wrote</td>
      <td>Generating an essay and submitting it as yours</td>
    </tr>
    <tr>
      <td>Summarizing a long source then reading the essentials</td>
      <td>Relying on the summary without ever reading the source</td>
    </tr>
