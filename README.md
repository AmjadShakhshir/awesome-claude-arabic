## ✨ لماذا هذا المستودع؟

مع انتشار أدوات مثل ChatGPT و Claude، الكثير من المستخدمين العرب لا يعرفون كيف يستفيدون منها بشكل عملي.

هذا المستودع يجمع:
- برومبتات جاهزة وقابلة للتعديل
- شروحات عملية بدون تعقيد
- طرق استخدام حقيقية (Workflows)
- أدوات مفيدة تم اختبارها

الهدف: مساعدتك تستخدم الذكاء الاصطناعي في شغلك، دراستك، أو مشاريعك بشكل فعّال.
